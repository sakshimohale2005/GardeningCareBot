🌱 Gardening Care Bot

Smart assistant for plant care

Description

This Gardening Care Bot is an intent-based chatbot that helps users with plant care. It provides guidance on watering, sunlight, soil, repotting, pruning, pests, indoor plants, fertilizing, and disease management using keyword-based responses.

Objectives

  • Provide instant plant care solutions
  • Help users maintain healthy plants
  • Answer common gardening questions

Intents Used

  • Sunlight
  • Soil
  • Repotting

  • Pests
  • Indoor Plants
  • General Help
  • Fertilizing
